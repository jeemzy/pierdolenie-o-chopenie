# 1. Ile lat trwa nauka i jakie są egzaminy po 9 klasie
```
W Białorusi szkoła trwa 11 lat.
Po 9 klasie:
- egzaminy: język białoruski, rosyjski, matematyka, historia Białorusi;
- wybór: zostać w szkole albo iść do koledżu, liceum lub technikum.
Wyniki wpływają na dalszą naukę.
```

# 2. „Dziennik” (dziennik lekcyjny) i „żurnał” (rejestr ocen i obecności)
```
„Dziennik” i „żurnał” służą do zapisu tematów, ocen, frekwencji i uwag.
Są ważnym narzędziem porządku, kontroli i nadzoru nad uczniem.
```

# 3. „Kontrolne” (sprawdziany) i kartkówki
```
„Kontrolne” i kartkówki stale sprawdzają wiedzę i dyscyplinę.
Często nie można ich poprawiać, więc słaba ocena długo wpływa na wynik.
```

# 4. Stołówka i jedzenie
```
Stołówka to ważna część dnia.
Jedzenie jest proste i tanie: zupa, kotlet, puree, kasza lub makaron, surówka, kompot.
Liczy się sytość, nie smak.
```

# 5. BRSM
```
BRSM to oficjalna organizacja młodzieżowa obecna w szkołach.
Jedni widzą w niej aktywność i możliwości, inni — ideologię i propagandę.
```

# 6. Zbiórka makulatury i rywalizacja klas
```
Zbiórka makulatury ma znaczenie ekologiczne, ale często zamienia się w rywalizację klas.
Staje się więc także źródłem presji.
```

# 7. Mundurek szkolny
```
Mundurek ma symbolizować porządek i dyscyplinę.
Wielu uczniów uważa jednak, że ogranicza indywidualność.
```

# 8. Makulatura i praca latem
```
W niektórych szkołach za słabą zbiórkę makulatury grozi „odrabianie” latem.
To pokazuje, że akcja bywa obowiązkiem i formą presji.
```

# 9. Osobne przedmioty językowe i historyczne
```
Przedmioty są rozdzielone:
- język rosyjski / literatura rosyjska;
- język białoruski / literatura białoruska;
- historia Białorusi / historia powszechna.
To porządkuje naukę.
```

# 10. Prodłużonka i sen po lekcjach
```
Młodsze dzieci mogą zostawać na prodłużonce po lekcjach.
Czasem mają tam miejsce do odpoczynku, a nawet snu.
To ułatwia opiekę i organizację dnia.
```
